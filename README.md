# Voyager

Track and manage your events and moments with voyager web application . 

Key Features :
- Dashboard - Data tracked is visualized into charts, graphs and tables.
- Moment Manager - Add/Delete/Edit data of moment tracked.
- Event Manager - Add/Delete/Edit data of events tracked.
- Filter - powerful Filter to search events or moments within a given time range.
